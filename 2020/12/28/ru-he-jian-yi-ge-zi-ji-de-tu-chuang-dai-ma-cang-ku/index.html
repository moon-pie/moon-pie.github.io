如何建一个自己的图床/代码仓库

发表于2020-12-28|更新于2021-05-03|技术
|阅读量:

先在github上创一个仓库
然后在本地新建一个文件夹并进入
克隆： git clone (仓库网址) https://github.com/Moon-Pie/code.git
然后进入克隆出来的文件夹：cd code

先将文件加入该文件夹
先输入初始化命令：git init
然后：git add .
上传：git commit -m “文件名”
push：git push
最后在博文中加入 /code/文件名.cpp 即可
