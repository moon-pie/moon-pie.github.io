完美串

发表于2021-04-22|更新于2021-05-03|思维题
|阅读量:

题意

给你一个长度为 n(n≤1000)n(n\le 1000)n(n≤1000) 的仅包含 ′0′,′1′,′?′'0','1','?'′0′,′1′,′?′ 的字符串。定义循环子串为将字符串首尾相接得到的圆的一段。定义一个完美串的所有相同长度的循环子串所包含的 ′1′'1'′1′ 的个数的极差至多为 111。请问将所有 ′?′'?'′?′ 替换成 ′0′'0'′0′ 或 ′1′'1'′1′ 所能得到的完美串的个数。

题解

一个串是完美串的必要条件：

被 ′1′'1'′1′ 划分出的 ′0′'0'′0′ 块的块长的极差 ≤1\le 1≤1。

证明十分显然，如果不满足这个条件，则一定有这样的串：

10…0⏟m100…00⏟m+2111\underbrace{0\dots0}_{m}1\underbrace{00\dots00}_{m+2}11 1m0…0​​1m+200…00​​11

左边长度为 mmm 的 ′0′'0'′0′ 块加上左右的 ′1′'1'′1′ 就会得到一个长度为 m+2m+2m+2 包含两个 ′1′'1'′1′ 的串，与完美串的定义矛盾。

因此一个完美串中的 ′0′'0'′0′ 块可以分成两种：长度为 mmm 的和长度为 m+1m+1m+1 的。

但这并不是充分条件，因为如果考虑一个长为 mmm 的串贡献为 000 ，长为 m+1m+1m+1 的串贡献是 111 ，由于其贡献存在差异，所以不能保证拼在一起还是完美串。反例：

0010100‾ 1010101‾\underline{0010100}\, \underline{1010101} 0010100​1010101​

但是，仔细一想可以发现贡献为 000 和贡献为 111 的串的拼接组成了一个子问题，也就是说这还需要是一个完美串。

如果我们设现在的串长 len\mathrm{len}len，并且有 x\mathrm{x}x 个 ′1′'1'′1′。设解下来的 ′0′'0'′0′ 块长为 m,m+1\mathrm{m,m+1}m,m+1，分别有 p,qp,qp,q 个，则有：

p⋅m+q⋅(m+1)=len−xp+q=x\begin{aligned} p\cdot m+q\cdot (m+1) &=len-x\\ p+q &= x \end{aligned} p⋅m+q⋅(m+1)p+q​=len−x=x​

现在我们来表演2个方程解3个未知数：

变形：

(p+q)⋅(m+1)=len−x+p\begin{aligned} (p+q)\cdot (m+1) = len-x+p\\ \end{aligned} (p+q)⋅(m+1)=len−x+p​

带入得到：

(m+1)⋅x=len−x+pm+1=len−x+px\begin{aligned} (m+1)\cdot x &= len-x+p\\ m+1 &= \frac{len-x+p}{x}\\ \end{aligned} (m+1)⋅xm+1​=len−x+p=xlen−x+p​​

由于 m+1m+1m+1 一定是整数，且 p≤xp\le xp≤x，所以，

m=⌈len−xx⌉−1m = \left\lceil\frac{len-x}{x}\right\rceil-1 m=⌈xlen−x​⌉−1

我的垃圾暴力推导：

下一个01串的长度为 len−x\mathrm{len-x}len−x，解方程得到 q=len−x−x(⌈len−xx⌉−1)\mathrm{q=len-x-x(\left\lceil\frac{len-x}{x}\right\rceil-1)}q=len−x−x(⌈xlen−x​⌉−1) 。

当 x∤len\mathrm{x\not\mid len}x∣len 时，q=(len−x) mod x=len mod x\mathrm{q = (len-x)\bmod x = len\bmod x}q=(len−x)modx=lenmodx ；
当 x∣len\mathrm{x\mid len}x∣len 时，总 0 数被总块数整除，所以剩下的都是 0。

考虑另一种 jasonfan{\color{black}\mathrm{j}}{\color{Red}\mathrm{asonfan}}jasonfan 的更巧妙推导方法：
考虑，

len mod x\begin{aligned} \mathrm{len \bmod x}\\ \end{aligned} lenmodx​

等于，

(len−q+q) mod x=(len−q) mod x+q mod x\begin{aligned} &\mathrm{(len-q+q)}\bmod x\\ =&\mathrm{(len-q)\bmod x+q\bmod x} \end{aligned} =​(len−q+q)modx(len−q)modx+qmodx​

len−q\mathrm{len-q}len−q 相当于把长串都减少 111 位，变成短串和 111 显然被 x\mathrm{x}x 整除。
然后，p+q=x⇒q≤x\mathrm{p+q=x\Rightarrow q\le x}p+q=x⇒q≤x，所以 q=len mod x\mathrm{q = len\bmod x}q=lenmodx。

然后就是类似 gcd⁡\gcdgcd 的递归，直到 x=0\mathrm{x=0}x=0 全部返回 000 。

可以发现，长度为 nnn 时，这样可以构造出 nnn 个完美串，每个完美串有 nnn 个循环，由于可能同构所以 完美串数≤n2\text{完美串数}\le n^2完美串数≤n2。

bitset\mathrm{bitset}bitset 优化判断是否可以和给出的字符串匹配就可以通过本题。
