基础莫队

学习笔记

还是先记一记几个比较基础的莫队的要点吧！

  • 普通莫队卡常可以用奇偶优化，就是奇块右端点的排序与偶块有端点的排序相反。

  • 树上莫队，由于一个节点第一次遇到是加，第二次遇到是减，可以先记录一个点是否遇到过，然后加操作与减操作就没有区别了。

  • 带修莫队，多了“时间”这一维，块长分为 n23n^{\frac{2}{3}}n32​，复杂度为 O(n53)O(n^{\frac{5}{3}})O(n35​)。一次Modify要记录修改前的状态和修改后的状态。

  • 回滚莫队，有只删不增和只增不删两种，左端点修改时需要记录这次修改的位置、修改前桶里的值、修改前的答案，并存到一个栈里，计算下一个询问之前回滚。注意有时候要对左右端点在同一块内的询问暴力计算。

二次离线莫队

问题

莫队二次离线解决这一类问题：

长度为 nnn 的序列，每次询问一段区间 [l,r][l,r][l,r] 中满足某种条件的点对的个数。

算法流程

区间 [l,r][l,r][l,r] 中与 xxx 满足条件点对的个数记为 f([l,r],x)f([l,r],x)f([l,r],x)。

考虑有端点从 rrr 移动到 r′r'r′ 答案的增量：

f([l,r],r+1)+f([l,r+1],r+2)+⋯+f([l,r′−1],r′)=∑i=r+1r′f([l,i−1],i)\begin{aligned} &f([l,r],r+1) + f([l,r+1],r+2)+\dots+f([l,r'-1],r')\\ =&\sum_{i=r+1}^{r'}f([l,i-1],i) \end{aligned} =​f([l,r],r+1)+f([l,r+1],r+2)+⋯+f([l,r′−1],r′)i=r+1∑r′​f([l,i−1],i)​

由于是点对的数量，fff 满足差分的性质，即有

f([l,r],x)=f([1,r],x)−f([1,l−1],x)f([l,r],x) = f([1,r],x) - f([1,l-1],x) f([l,r],x)=f([1,r],x)−f([1,l−1],x)

那么答案的增量就是

∑i=r+1r′f([1,i−1],i)−∑i=r+1r′f([1,l−1],i)\sum_{i = r+1}^{r'}f([1,i-1],i) - \sum_{i=r+1}^{r'}f([1,l-1],i) i=r+1∑r′​f([1,i−1],i)−i=r+1∑r′​f([1,l−1],i)

左边的可以提前预处理出来，右边的式子再离线下来统一询问（这就是所谓二次离线）。

同理可得：

当左端点从 lll 移动到 l′l'l′，答案的增量为

f([l,r],l−1)+f([l−1,r],l−2)+⋯+f([l′+1,r],l′)=∑i=l′l−1f([i+1,r],i)=∑i=l′l−1f([i+1,n],i)−∑i=l′l−1f([r+1,n],i)\begin{aligned} &f([l,r],l-1) + f([l-1,r],l-2)+\dots+f([l'+1,r],l')\\ =&\sum_{i=l'}^{l-1}f([i+1,r],i)\\ =&\sum_{i = l'}^{l-1}f([i+1,n],i) - \sum_{i=l'}^{l-1}f([r+1,n],i)\\ \end{aligned} ==​f([l,r],l−1)+f([l−1,r],l−2)+⋯+f([l′+1,r],l′)i=l′∑l−1​f([i+1,r],i)i=l′∑l−1​f([i+1,n],i)−i=l′∑l−1​f([r+1,n],i)​

当右端点向左移动到 r′r'r′，减少量为

−f([l,r−1],r)−f([l,r−2],r−1)−⋯−f([l,r′],r′+1)=−∑i=r′+1rf([l,i−1],i)=−∑i=r′+1rf([1,i−1],i)+∑i=r′+1rf([1,l−1],i)\begin{aligned} &-f([l,r-1],r) - f([l,r-2],r-1)-\dots-f([l,r'],r'+1)\\ =&-\sum_{i=r'+1}^{r}f([l,i-1],i)\\ =&-\sum_{i = r'+1}^{r}f([1,i-1],i) + \sum_{i=r'+1}^{r}f([1,l-1],i) \end{aligned} ==​−f([l,r−1],r)−f([l,r−2],r−1)−⋯−f([l,r′],r′+1)−i=r′+1∑r​f([l,i−1],i)−i=r′+1∑r​f([1,i−1],i)+i=r′+1∑r​f([1,l−1],i)​

当左端点向右移动到 l′l'l′，减少量为

−f([l+1,r],l)−f([l+2,r],l+1)−⋯−f([l′,r],l′−1)=−∑i=ll′−1f([i+1,r],i)=−∑i=ll′−1f([i+1,n],i)+∑i=ll′−1f([r+1,n],i)\begin{aligned} &-f([l+1,r],l)-f([l+2,r],l+1)-\dots-f([l',r],l'-1)\\ =&-\sum_{i=l}^{l'-1}f([i+1,r],i)\\ =&-\sum_{i=l}^{l'-1}f([i+1,n],i) + \sum_{i=l}^{l'-1}f([r+1,n],i)\\ \end{aligned} ==​−f([l+1,r],l)−f([l+2,r],l+1)−⋯−f([l′,r],l′−1)−i=l∑l′−1​f([i+1,r],i)−i=l∑l′−1​f([i+1,n],i)+i=l∑l′−1​f([r+1,n],i)​

不难发现我们二次离线下来的询问个数为 O(nn)O(n\sqrt n)O(nn​)，因为左右端点移动次数为 O(nn)O(n\sqrt n)O(nn​) 次。

二次离线下来的询问的形式是 f([1,l],x)f([1,l],x)f([1,l],x) 就是一个前缀和一个数组成的满足条件的点对的个数。

第十四分块(前体)

问题是，询问一段区间 [l,r][l,r][l,r] 中，ai⊕aja_i\oplus a_jai​⊕aj​ 在二进制表示下有 kkk 个 111 的点对 (i,j)(i,j)(i,j) 个数，0≤ai<163840\le a_i < 163840≤ai​<16384。

直接考虑 O(nn)O(n\sqrt n)O(nn​) 个形如 f([1,l],x)f([1,l],x)f([1,l],x) 的询问如何处理。

貌似直接在值域上搞一搞也就可以了，时间复杂度 O(nn+n(kk/2))O(n\sqrt n + n\binom{k}{k/2})O(nn​+n(k/2k​))。
