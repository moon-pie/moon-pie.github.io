SAM 的定义

  • SAM 是一张有向无环图。结点被称作 状态，边被称作状态间的 转移。
  • 图存在一个源点 t0t_0t0​，称作 初始状态，其它各结点均可从 t0t_0t0​ 出发到达。
  • 每个 转移 都标有一些字母。从一个结点出发的所有转移均 不同。
  • 存在一个或多个 终止状态。如果我们从初始状态 t0t_0t0​ 出发，最终转移到了一个终止状态，则路径上的所有转移连接起来一定是字符串 sss 的一个后缀。sss 的每个后缀均可用一条从 t0t_0t0​ 到某个终止状态的路径构成。
  • 在所有满足上述条件的自动机中，SAM 的结点数是最少的。

在线性时间内构造 SAM

结束位置 endpos

对于字符串 sss 的一个子串 ttt，endpos(t)\mathrm{endpos}(t)endpos(t) 就是 ttt 在 sss 中所有结束位置的集合。

  • SAM 中每个状态都对应一个或多个 endpos\mathrm{endpos}endpos 相同的子串。
  • 字符串 sss 的两个非空子串 u,wu,wu,w，假设 ∣u∣≤∣w∣|u| \le |w|∣u∣≤∣w∣ 的 endpos\mathrm{endpos}endpos 相同，当且仅当 uuu 在 sss 中每次出现都是以 www 的后缀的形式。
  • 字符串 sss 的两个非空子串 u,wu,wu,w，假设 ∣u∣≤∣w∣|u| \le |w|∣u∣≤∣w∣ 。
    要么 endpos(w)∩endpos(u)=∅\mathrm{endpos(w)\cap \mathrm{endpos}(u) = \varnothing}endpos(w)∩endpos(u)=∅ ，要么 endpos(w)⊆endpos(u)\mathrm{endpos}(w) \subseteq \mathrm{endpos}(u)endpos(w)⊆endpos(u)
    （没交，或长串是短串子集，因为长串出现的次数少）。
  • 对于一个状态 vvv，有一个或多个子串对应与之。记其中最长的子串的长度为 len(v)\mathrm{len}(v)len(v)，最短的子串长度为 minlen(v)\mathrm{minlen}(v)minlen(v)。那么对应于这个状态的字符串都是字符串 longest(v)\mathrm{longest}(v)longest(v) 的不同的后缀，且所有字符串的长度恰好覆盖 [minlen(v),len(v)][\mathrm{minlen}(v),\mathrm{len}(v)][minlen(v),len(v)] 的每一个整数。

后缀链接 link

我们已经知道一个状态 vvv 对应于具有相同 endpos\mathrm{endpos}endpos 的子串。如果我们定义最长的那个为 www 则其他所有字符串都是 www 的后缀。

那么，后缀链接 link(v)\mathrm{link}(v)link(v) 连接到 www 的最长的、且 endpos\mathrm{endpos}endpos 与 www 不同的后缀对应的状态。

  • 所有后缀链接构成一棵根节点为 t0t_0t0​ 的树。
